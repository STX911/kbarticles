TR Knowledgebase: Structure Fire

Structure / Apartment Fire
Useful Information

Attempt to obtain a list of occupants / number of apartments

Enquire if there are any mentally ill or physically handicapped occupants.

Have there been any previous calls to this address?

  • If so, is there a pattern and previous damage?

Consider access to hydrant locations or nearby water sources

Is there any potential high life threat?
